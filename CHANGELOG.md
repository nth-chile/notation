# Changelog

All notable changes to Nubium are listed here. Newest first.

## Unreleased

## 0.1.3 — 2026-04

- Getting Started help modal and onboarding affordances for new users.
- Undo snapshots now cover selection-based articulation, accidental, and delete edits.
- Optional email field on the in-app feedback form so we can reply.
- "What's new" modal pops on first launch after an update; also accessible from Help menu.
- Public changelog page on the website.
- Outbound JACK transport sync (Linux only): Nubium's playback drives connected JACK clients.

## 0.1.1 — 2026-04

- Tab-stave parity: fill indicator, auto-advance, digit gate, click-to-nearest-string.
- Cursor stays in place when on a rest in tab-only view.
- Repeat-count polish, tab parity improvements, chord tie fixes.
- First-system indent only when part labels will be shown.
- Bumped tab fret font to 11pt and measure-number font to 13px.
- Redrew document icon using website SVG logo.
- Windows release bundles NSIS only (skips MSI).

## 0.1.0 — 2026-04

- First public beta.
- Section breaks reset measure numbering; Radix Select used in Parts panel.
- Replaced emoji break markers on canvas with blue-stroked icons.
- Settings polish; telemetry adds platform tracking, opt-out removed.
- Telemetry routed to stats.nubium.rocks with HTML dashboard.
- Transposing playback preview, refined color palette, MusicXML mid-measure key changes.
- Desaturated canvas accents to match parchment warmth.
- Primary color set to #3b82f6; muted notes use color mixing.
- Toolbar button flash ring no longer clips.
